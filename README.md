# UKFITDB3 Research Publication Finder

A minimal React boilerplate with Vite, Vitest, and Testing Library.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Setup the Local Environment

1. Clone or navigate to the project directory:

   ```bash
   cd react-hello-world
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

## Run Tests

Run all tests once:

```bash
npm run test:run
```

Run tests in watch mode (re-runs on file changes):

```bash
npm run test
```

## Lint Code

Run ESLint with React rules to check for code quality issues:

```bash
npm run lint
```

The CI pipeline automatically runs lint checks on every push and pull request.

## Run Local Dev Server

Start the development server with hot module replacement:

```bash
npm start
```

The app will be available at `http://localhost:5173`.

## Build for Production

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
